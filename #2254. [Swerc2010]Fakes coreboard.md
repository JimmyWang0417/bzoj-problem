
### Description
T个队在做P个题。现在给出每个队解决题的个数，及每个题被通过的次数。
问这个情况是否存在，如果存在输出字典序最小的一个解，不存在输出"Impossible"
### Input
第一行给出T和P，1<=T,P<=80。
第二行T个数字，其值在0到90,互不相同.
第三行P个数字，代表P个题各有多少队通过了.
本题有多组测试数据，做到"0 0"结束
### Output
如题
### Sample Input
2 2
1 2
1 1

3 3
2 1 2
1 2 2

3 5
3 3 1
3 1 1 0 2

0 0

### Sample Output
Impossible

NYY
NNY
YYN

YNYNY
YYNNY
YNNNN
