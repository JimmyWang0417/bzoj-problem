
### Description
从{0,…,N-1}这N个数字中选K个数字总共有多少种方案？,显然有N!/K!/(N-K)!种方案。在这N!/K!/(N-K)!种方案有多少种方案，它们的数字和(不是数位和)能被N整除？ 
 

### Input
      第一行包括两个整数N,K
 

### Output
      一个整数，为方案数mod 1000000007（质数）后的值
 

### Sample Input
7 4


### Sample Output
5

### Hint
总共是以下5种方案：
{0,1,2,4}{0,3,5,6}{1,2,5,6}{1,3,4,6}{2,3,4,5}
 
100%的数据满足1<=N<=10^9,1<=K<=1000
