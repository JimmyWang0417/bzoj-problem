
### Description
 有一个大小为n的可重集S，小奇每次操作可以加入一个数a+b(a,b均属于S)，求k次操作后它可获得的S的和的最大
值。（数据保证这个值为非负数）
### Input
第一行有两个整数n,k表示初始元素数量和操作数，第二行包含n个整数表示初始时可重集的元素。
对于100%的数据，有 n<=10^5，k<=10^9，|ai|<=10^5
### Output
输出一个整数，表示和的最大值。答案对10000007取模。


### Sample Input
2 2
3 6

### Sample Output
33
