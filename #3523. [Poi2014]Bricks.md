
### Description
有n种颜色的砖块，第i种颜色的砖块有a[i]个，你需要把他们放成一排，使得相邻两个砖块的颜色不相同，限定第一个砖块的颜色是start，最后一个砖块的颜色是end，请构造出一种合法的方案或判断无解。

### Input
第一行3个数，n，start，end。
第二行n个数，a[i]。

### Output
令m=sigma(a[1..n])。
如果有解输出m个数。
无解输出0。

### Sample Input
3 3 1
2 3 3


### Sample Output
3 2 1 3 2 3 2 1


### Hint
【数据范围】
n,m≤1000000，1≤start,end≤n
