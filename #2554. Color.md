
### Description
有n个球排成一列，每个球都有一个颜色，用A-Z的大写字母来表示，我们每次随机选出两个球ball1,ball2,使得后者染上前者的颜色，求期望操作多少次，才能使得所有球的颜色都一样？

### Input
    一行一个字符串，表示球的颜色
   
### Output
 一行表示结果，精确到小数点后1位。
   
### Sample Input
   AAA

### Sample Output
   0.0
### Hint
数据范围
   对于10%的数据，n<=20
   
   对于40%的数据，n<=200
   
   对于50%的数据，n<=1000
   
   对于100% 的数据，n <= 10000
