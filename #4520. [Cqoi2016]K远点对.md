
### Description
已知平面内 N 个点的坐标，求欧氏距离下的第 K 远点对。

### Input
输入文件第一行为用空格隔开的两个整数 N， K。接下来 N 行，每行两个整数 X,Y，表示一个点
的坐标。1 < =  N < =  100000， 1 < =  K < =  100， K < =  N*(N−1)/2 ， 0 < =  X, Y < 2^31。

### Output
输出文件第一行为一个整数，表示第 K 远点对的距离的平方（一定是个整数）。

### Sample Input
10 5
0 0
0 1
1 0
1 1
2 0
2 1
1 2
0 2
3 0
3 1
### Sample Output
9
