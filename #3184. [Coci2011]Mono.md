
### Description
给出一个由小写字母组成的R*C的矩阵。在这个矩阵中，有一个边平行于坐标
轴的V个顶点的简单多边形。多边形的顶点按顺时针依次给出。你可以将多边形上
下左右任意移动。问有多少个能够由给出的多边形移动得到的多边形，其内部仅含
有一种字母。 
 
 
### Input
第一行给出R，C
接下来R行C列
接下来给出数字V
接下来V行，每行两个数描述顶点坐标
### Output
如题
### Sample Input
3 3 
aaa 
aaa 
aaa 
4 
2 0 
2 2 
0 2 
0 0 
### Sample Output
4
### Hint
R, C, V <= 500.
