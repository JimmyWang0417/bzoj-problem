
### Description
　　阿申准备报名参加GT考试，准考证号为N位数X1X2....Xn(0<=Xi<=9),他不希望准考证号上出现不吉利的数字。
他的不吉利数学A1A2...Am(0<=Ai<=9)有M位，不出现是指X1X2...Xn中没有恰好一段等于A1A2...Am. A1和X1可以为
0
### Input
　　第一行输入N,M,K.接下来一行输入M位的数。 N<=10^9,M<=20,K<=1000
### Output
　　阿申想知道不出现不吉利数字的号码有多少种，输出模K取余的结果.
### Sample Input
4 3 100 
111
### Sample Output
81
