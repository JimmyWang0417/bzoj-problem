
### Description

![](/JudgeOnline/upload/201701/11.jpg)
令Ei=Fi/qi，求Ei.

### Input

第一行一个整数n。
接下来n行每行输入一个数，第i行表示qi。
n≤100000，0<qi<1000000000



### Output
