
### Description
给出一个天平，一个待称物体，重量为M，一些砝码，重量分别为1,3,9...3^K.
每个砝码只有一个 。
现在要求你把物体放在天平的左盘，然后在两个盘中放入一些砝码，使得天平平衡 。


### Input
一个整数M，M<=10^100


### Output
两行，第一行表示放在左盘的砝码，第二行表示放在右盘的砝码。每行的第一个数字表示
这个盘中放K人砝码，然后是连续的K个数字，按升序输出，表示这些砝码的重量 。

### Sample Input
42
### Sample Output
3 3 9 27
1 81
