
### Description
　　给出N，统计满足下面条件的数对(a,b)的个数：
　　1.1<=a<b<=N
　　2.a+b整除a*b


　

### Input
　一行一个数N

　
### Output
　一行一个数表示答案

### Sample Input
15

### Sample Output
4
