
### Description
![](/images/1988.jpg)

### Input
输入只有1行，4个整数，分别为 a,m,b,n。


### Output
从最高项到最低项输出多项式F的系数
ad, ad-1,…a1,a0

### Sample Input
样例输入1：
3 2 2 3


样例输出2：
2 2 3 4


### Sample Output
样例输出1：
1 0 -9 -4 27 -36 -23

样例输出2：
1 0 -8 0 18 0 -104 0 1

数据范围：
	t<=4
	m*n<=20
	保证答案F的系数范围在[-2^31+1，2^31-1]以内

### Hint
可以证明答案F的最高次数为n*m。
注意输出的格式，一切形如+2，+0，-0的输出均视为不合法输出。
