
### Description
给出一个平行四边形的长与高.
给出入口与出口,对于四边形的每条边都有一种Color,非黑即白.
你要找出一条路径,走过的边黑白交替,且长度最短
### Input
第一行给出平行四边形的长与高.且值在[1,500]
第二行给出入口,出口坐标.
下面的字符矩阵代表这个平行四边形每条边的Color
### Output
输出路径的最少长度
### Sample Input
inupt 1
2 1
0 0 2 1
bb
nwwnw
bn

input 2
5 4
0 2 5 2
nnbnn
nnnwwbwnnnn
nbbbn
nnwbwwbwwnn
bwwww
nnbwbbwwbnn
nwwwn
nnnnbwbbnnn
nnwnn
### Sample Output
output 1
6

output 2
22
