
### Description
　你有一个n*m的矩形，一开始所有格子都是白色，然后给出一个目标状态的矩形，有的地方是白色，有的地方是黑色，你每次可以选择一个连通块（四连通块，且不要求颜色一样）进行染色操作（染成白色或者黑色）。问最少操作次数。

### Input
　　第一行两个数n，m表示矩形大小。
　　接下来n行描述目标状态，每行m个字符，’W’表示白色，’B’表示黑色。

### Output
　　一行一个整数表示操作数。

### Sample Input
3 3
WBW
BWB
WBW


### Sample Output
2


### Hint
数据规模和约定
　　100%的数据n<=50,m<=50
　　15%的数据n*m<=15
　　另外15%的数据m=1
