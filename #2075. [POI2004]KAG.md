### Input
第一行一个整数k 表示k组数据, 1 <= k <= 10, 接下来描述每个数据开头两个数n 和 m, 1 <= n <= 10.000, 0 <= m <= 100.000. 接下来m行每行两个数表示一条边. 


### Output
k 行每行输出一个判定,如果该图为c-algae输出TAK,否则输出NIE. 

### Sample Input
3
3 2
1 2
2 3
4 3
1 2
2 3
3 4
3 3
1 2
2 3
3 1

### Sample Output
TAK
NIE
TAK
