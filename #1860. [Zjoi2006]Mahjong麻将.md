### Input
第一行一个整数N（N<=100），表示玩了N次超级麻将。
接下来N行，每行100个数a1..a100，描述每次玩牌手中各种牌的数量。ai表示数字为i的牌有ai张。（0<=ai<=100）


### Output
输出N行，若胡了则输出Yes，否则输出No，注意区分Yes，No的大小写！


### Sample Input
3
2 4 0 0 0 0 0 …… 0（一共98个0）
2 4 2 0 0 0 0 …… 0（一共97个0）
2 3 2 0 0 0 0 …… 0（一共97个0）

### Sample Output
Yes
Yes
No
