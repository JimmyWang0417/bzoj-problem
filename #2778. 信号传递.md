
### Description
 Berland的首都被n座山峰包围，形成一个圆圈。每座山的山顶都有一个瞭望塔，瞭望塔有高度的区别。如果存在连接两座山峰A和B的弧上没有严格比A或B高的塔，那么A和B可以直接传达信息。输入n以及n座塔的高度，输出有多少对瞭望塔可以直接互相传递信息。
### Input
n 
H1..Hn 
均为整数
 
 
### Output
一个整数，表示能直接传递信号的瞭望塔对数

### Sample Input
5
1 2 4 5 3

### Sample Output
7
### Hint
100%： 3 ≤ n ≤ 10^6     1 ≤ H ≤ l0^9
