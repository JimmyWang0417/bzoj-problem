
### Description
聪明的兔子定义了三个函数:
• g(x) 表示将x 十进制表示中各位数字从高位到低位降序排列组成的数字。
• L(x) 表示将 x 十进制表示中各位数字从高位到低位升序排列组成的数字。
• f(x)= g(x) -  L(x)
如果一个数x与f(x)相等，兔子们称它为幸运数。现在兔子们希望计算在所有的n位数字中，所有幸运
数的平方的和。结果对 p 取模后输出，注意这里数字可以包含前导 0.

### Input
一行两个整数n 和 p

### Output

一行一个整数，表示所求的结果

### Sample Input
4 10000

### Sample Output

8276
### Hint

对于 100% 的测试数据，1 < =  n < =  30; 1 < =  p < =  2 *10^9
