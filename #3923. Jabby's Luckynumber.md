
### Description

数字成为幸运数。现在它得到了两个位数相同的幸运数l,r。它设l,r 之间的幸运数为a1,a2...������an现在它想知道∑A(i)*A(i-1),1<=i<=N-1,对1000000007取模的结果。


### Input
第一行给定两个数a,b
第二行给出l
第三行给出r


### Output
如题所述


### Sample Input

474
777
### Sample Output
2316330
### Hint
