
### Description
已知数a,p,b，求满足a^x≡b(mod p)的最小自然数x。
 
### Input
    每个测试文件中最多包含100组测试数据。
    每组数据中，每行包含3个正整数a,p,b。
    当a=p=b=0时，表示测试数据读入完全。
 
### Output
 
    对于每组数据，输出一行。
    如果无解，输出“No Solution”（不含引号），否则输出最小自然数解。
 
### Sample Input
    5 58 33
    2 4 3
    0 0 0


### Sample Output
9
No Solution


### Hint
  100%的数据，a,p,b≤1e9。
