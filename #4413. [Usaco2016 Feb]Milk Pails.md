
### Description
有两个桶大小为X和Y，开始都是空的。
有如下三种操作，最多操作K次。
①用倒满一个桶。
②倒空一个桶。
③把一个桶里的水倒到另一个桶里，直到一个桶空了或者另一个桶满了。（看哪种情况先发生）

给出要求的水量M.
最后你获得的水量是两个桶的水量之和M'。
求|M-M'|的最小值。
### Input
一行，四个数X,Y,K,M.
1<=X,Y,K<=100
1<=M<=200
### Output
输出一个数表示最小值。
### Sample Input
14 50 2 32
### Sample Output
18
