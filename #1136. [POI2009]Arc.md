
### Description
给定一个序列{ai | 1 <= ai <= 1000000000 且 1 <= i <= n 且 n <= 15000000}和一个整数 k (k <= n 且 k <= 1000000)，求出a的一个长度为k的子序列{a[bi]}满足： (1) 1 <= b1 <= b2 <= ... <= bk (2) 在满足(1)的情况下 {a[b1], a[b2], ... , a[bk]} 字典序最大。
### Input
第一行一个数k，以下一行，为序列ai。以一个单独的0结束
### Output
k行，每行一个数，其中第i行为a[bi]。
### Sample Input
12
5 8 3 15 8 0

### Sample Output
12
15
8
