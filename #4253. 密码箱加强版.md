
### Description
由于BZOJ的水题已经被Lcomyn大爷刷完了，所以机房的蒟蒻们决定再出一道水题给lcomyn刷。
正巧蒟蒻们最近做了AHOI2007密码箱，觉得这题实在太水了。稍微加强一下下依然还是很水的，所以决定把数据范围开大一点送给Lcomyn切。
求方程x^2=1(mod n)的解,x为小于n的非负整数。

### Input
一行一个数n。

### Output
如果方程无解输出None,否则就按从小到大的顺序输出所有解，两个数之间用空格隔开，行末无多余空格。

### Sample Input
5
### Sample Output
1 4
### Hint
1<=n<=10^18
